Sistema Experto para la Detección de Embarazos y otras Patologías (SEDEP)

 Este sistema experto facilita la detección de embarazos y otras patologías gineco-obstétricas. El sistema utiliza una serie de reglas lógicas para poder determinar si una paciente está embarazada o puede estar sufriendo de alguna otra patología. Es una herramienta útil para los especialistas ginecológicos, ya que facilita la detección de patologías y reduce los tiempos de diagnóstico.
