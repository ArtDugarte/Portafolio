Portal API Mercantil

 En el contexto de mi trabajo en el Banco Mercantil de Venezuela, participé en la adaptación y actualización de una plantilla en el portal API Mercantil. Mi enfoque se centró en el frontend, utilizando HTML, CSS y JavaScript para realizar estas mejoras.

 La plataforma API Connect y el sistema de gestión de contenido Drupal, en sus versiones 9 y 10, fueron los pilares tecnológicos de este proyecto. A través de la implementación de estas herramientas, logramos modernizar la apariencia y funcionalidad del sitio web. Mi labor consistió en la manipulación y diseño de elementos visuales, asegurando una presentación estética y atractiva para los usuarios.

 El resultado final fue una interfaz renovada y amigable que no solo cumplió con los estándares modernos de diseño web, sino que también proporcionó una experiencia agradable a los visitantes del portal API Mercantil.
